git branch - показывает список локальных веток в вашем репозитории.

git branch <имя_ветки> - создает новую ветку с указанным именем.

git checkout <имя_ветки> - переключается на указанную ветку.

git checkout -b <имя_ветки> - создает новую ветку и переключается на нее.

git merge <имя_ветки> - объединяет указанную ветку с текущей веткой.

git push origin <имя_ветки> - отправляет локальную ветку в удаленный репозиторий.

git pull origin <имя_ветки> - получает и объединяет изменения из указанной ветки удаленного репозитория.

git tag - показывает список тегов в вашем репозитории.

git tag <имя_тега> - создает новый тег с указанным именем для текущего коммита.

git tag -a <имя_тега> -m <сообщение> - создает аннотированный тег с указанным именем и сообщением для текущего коммита.

git checkout <имя_тега> - переключается на указанный тег.

git push origin --tags - отправляет все теги в удаленный репозиторий.

git fetch --tags - получает все теги из удаленного репозитория.

git stash - сохраняет текущие незафиксированные изменения во временном хранилище.

git stash apply - применяет последние сохраненные изменения из временного хранилища.

git stash pop - применяет последние сохраненные изменения и удаляет их из временного хранилища.
